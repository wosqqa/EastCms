一、使用右键压缩

右键点击需要压缩的文件或文件夹，在弹出菜单中选择如何压缩。

  1. "创建压缩文件..."：先设置后压缩。
  2. "添加到'文件名.格式'"：按默认设置压缩（可修改，点击看详细）。
  3. "压缩并Email..."：先设置后压缩，并用Email发送。
  4. "压缩到'文件名.格式'并Email"：按默认设置压缩，并用Email发送。
