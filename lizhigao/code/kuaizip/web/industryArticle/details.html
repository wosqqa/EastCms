这样推广主页导航收入更多，新技能get

浏览次数：252|更新日期：2015/05/02 11:00

在主页导航的推广工程中，很多人技术员朋友都有这样一个认识：推广主页给用户，如果他们每天都使用的话，积分增加会很快，随之收入增加也会很快。所以我们要尽量避免主页的流失，避免主页导航被篡改。

1.用安全管理软件—卫士、管家等锁定主页；

2.安装其他浏览器，他们也会锁定一个主页，可能锁定的和你要推广的不一样，这时要注意手动修改浏览器锁定的其他主页；比如360浏览器；

3.装机联盟支持双导航，可以在不同的浏览器上设置不同的主页导航。
