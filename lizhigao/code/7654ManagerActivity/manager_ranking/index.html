活动规则

Q：这个活动是所有人都能参加吗？

A：这个活动仅限正式市场经理参加。



Q：榜单奖励多久给一次？

A：每月排名一次，当月数据随着业绩的更新逐步更新，最终排名以当月推广业绩更新完毕时为准。



Q：两个榜单是什么意思咧？

A：每月按照两个要素分别进行排名，只要满足任一榜单前20名即可获得奖励。



Q：两个榜单都有我，那我能拿两份奖励吗？

A：最终奖励依据排名较高的榜单哦。



Q：下属推广业绩，推广含哪些产品呢？

A：7654推广资源里面的所有软件产品。



Q：活动按照有业绩且资料真实完整的技术员人数排名，但是为什么排行榜中展示的是有业绩且资料完整的技术员人数呢？

A：活动是按照当月有业绩且资料真实完整的技术员人数排名的，排行榜中展示的资料完整的技术员人数，其真实性7654需要进行人工审核，最终人数可能与列表中的不一致哦。



Q：活动奖励什么时候发放呢？

A：活动奖励在本月业绩更新完毕时准备发放，预计在次月15日左右发放至账户可用积分。
