Q：今天安装的什么时候给我返工资？

A：今天安装的，明日17：00左右会返回安装量和工资到账户，周末、国家法定假日顺延至工作日返回。


Q：返回给我的工资什么时候能提现？

A：本月的工资次月15日便可提现。


Q：我自己安装的快压会算入我的收入吗？

A：算有效的。


Q：推广业绩如何计算？

A：当用户下载您用快压压缩的资料后，通过引导使用快压成功打开资料后，则为推广成功，但也需要满足以下条件：

  1. 1、联网情况下安装成功，即为有效；
  2. 2、用户电脑3月内安装过该软件，则覆盖安装或重复安装无效；
  3. 3、网吧机房环境下安装无效；
  4. 4、系统封装无效；


Q：我的收入怎么提现？

A：收入采用提取到支付宝账户的方式提现。


Q：提现后多长时间能拿到钱？

A：提现后5个工作日内完成审核并转账。


Q：快压的官方包更新的情况下，我这边需要重新下载包吗？

A：需要重新下载安装包的。
